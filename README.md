# time-input with vanilla js
### What is this ?
add `hh:mm`format to input element.

### how to use
1. install 
```
npm install time-inputter
```

2. add import
```js
import time-inputter from 'time-inputter';
```

3. add `time-inputter` to className
```html
<input class="time-inputter" value="09:00" />
```
